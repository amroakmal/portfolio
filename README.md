# Amr Abouelmagd — Portfolio

Personal academic portfolio: HPC & GPU systems research, publications, experience, and projects.

**Live:** https://amroakmal.github.io/portfolio/

Single self-contained `index.html` (no build step). Served directly by GitHub Pages — the
`.nojekyll` file disables Jekyll processing so the HTML is served as-is.

## Editing
Everything lives in `index.html` — edit the text directly. When your project
repos are public, you can link a card by wrapping its `<div class="proj">` in an
`<a href="...">` tag.
